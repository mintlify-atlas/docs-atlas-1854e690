MMAction2 is an open-source video understanding toolbox built on PyTorch. It is part of the [OpenMMLab](https://openmmlab.com) project and provides a unified framework for training, evaluating, and deploying state-of-the-art models across multiple video understanding tasks — from clip-level action recognition to frame-level spatio-temporal detection.

## What MMAction2 supports

MMAction2 covers five major video understanding tasks out of the box:

- **Action recognition** — classify actions in trimmed or untrimmed video clips
- **Temporal action localization** — detect when actions occur in long, untrimmed videos
- **Spatio-temporal action detection** — localize who is doing what and where, frame by frame
- **Skeleton-based action recognition** — recognize actions from human pose keypoint sequences
- **Video retrieval** — retrieve semantically relevant videos using vision-language models

## Major features

**Modular design.** MMAction2 decomposes each video understanding pipeline into independently configurable components — backbones, necks, heads, data pipelines, and losses. You can swap or extend any component without touching the rest of the framework.

**Multi-task support.** A single codebase covers recognition, localization, spatio-temporal detection, skeleton-based recognition, and video retrieval. Shared infrastructure such as the config system, data loading, and training loops works consistently across all tasks.

**Rich tooling.** The toolbox ships with analysis and visualization scripts including dataset browsers, learning rate schedulers, model profilers, and benchmark evaluation utilities to help you debug, tune, and compare models.

**OpenMMLab integration.** MMAction2 follows OpenMMLab's interface conventions and registry system, so code from other OpenMMLab libraries — MMDetection, MMPose, MMClassification — can be called directly. Switching between projects requires minimal relearning.

## OpenMMLab ecosystem

MMAction2 is built on top of two foundational OpenMMLab libraries:

- **[MMEngine](https://github.com/open-mmlab/mmengine)** — the training engine providing the `Runner`, config inheritance, hooks, logging, and distributed training abstractions.
- **[MMCV](https://github.com/open-mmlab/mmcv)** — the computer vision library providing image/video transforms, model layers, and CUDA ops.

Optional integration with **[MMDetection](https://github.com/open-mmlab/mmdetection)** and **[MMPose](https://github.com/open-mmlab/mmpose)** enables spatio-temporal detection and skeleton-based recognition pipelines that chain detection, pose estimation, and action recognition in a single workflow.

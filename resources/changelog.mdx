MMAction2 follows a structured release cycle. The 1.x series, built on [MMEngine](https://github.com/open-mmlab/mmengine), is the actively maintained branch. The legacy 0.x series (the `master` branch) is in maintenance mode. New users should start with 1.x.

<Note>
  The default branch was switched from `master` (0.x) to `main` (1.x). If you are migrating an existing project, see the [migration guide](https://mmaction2.readthedocs.io/en/latest/migration.html).
</Note>

## v1.2.0 — October 12, 2023

This release adds multi-modality model training, lightweight backbone support, a new video retrieval dataset, and improved demo tooling.

**Highlights**

- ActionCLIP training support (training was inference-only in v1.1.0)
- VindLU multi-modality video-language algorithm
- MobileOne backbone for TSN and TSM (lightweight models)

### New features

- **ActionCLIP training** — Full training support for the ActionCLIP model ([#2620](https://github.com/open-mmlab/mmaction2/pull/2620))
- **VindLU** — Multi-modality video-language algorithm ([#2667](https://github.com/open-mmlab/mmaction2/pull/2667))
- **MSVD dataset** — Video retrieval dataset support ([#2622](https://github.com/open-mmlab/mmaction2/pull/2622))
- **Dense Regression Network** — Video grounding with DRN ([#2668](https://github.com/open-mmlab/mmaction2/pull/2668))

### Improvements

- MobileOne TSN/TSM backbones for efficient inference ([#2656](https://github.com/open-mmlab/mmaction2/pull/2656))
- SlowOnly K700 features for training temporal localization models ([#2673](https://github.com/open-mmlab/mmaction2/pull/2673))
- Video demo support ([#2602](https://github.com/open-mmlab/mmaction2/pull/2602))
- Audio demo support ([#2603](https://github.com/open-mmlab/mmaction2/pull/2603))

### Bug fixes

- Refine `ActionDataSample` structure ([#2658](https://github.com/open-mmlab/mmaction2/pull/2658))
- Fix MPS device compatibility ([#2619](https://github.com/open-mmlab/mmaction2/pull/2619))

## v1.1.0 — July 3, 2023

A major feature release focused on new datasets, CLIP-based multi-modality models, and knowledge distillation.

**Highlights**

- Three new datasets: HACS-segments (ICCV 2019), MultiSports (ICCV 2021), and Kinetics-710
- Gesture recognition and spatio-temporal detection projects
- VideoMAE V2 (CVPR 2023) and VideoMAE on action detection
- ActionCLIP and CLIP4Clip multi-modality models
- Pure Python-style configuration file support

## v1.0.0rc0 — September 1, 2022

The first release in the 1.x series, built on the new MMEngine training engine.

<Warning>
  This release introduced **breaking changes** in dataset interfaces, config structure, model APIs, and evaluation. See the [migration guide](https://mmaction2.readthedocs.io/en/latest/migration.html) for instructions on porting 0.x configs and code.
</Warning>

Key changes from 0.x:

- Replaced the custom runner with MMEngine's unified runner
- Unified train/test/dataset/evaluation/visualization interfaces across all OpenMMLab 2.0 projects
- Significantly refactored configuration file structure

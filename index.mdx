MMAction2 is an open-source video understanding toolbox built on PyTorch and part of the [OpenMMLab](https://openmmlab.com) ecosystem. It provides a unified framework for training and evaluating state-of-the-art models across five core video understanding tasks: action recognition, temporal action localization, spatio-temporal action detection, skeleton-based action recognition, and video retrieval.

## Get started in 3 steps

<Steps>
  <Step title="Install MMAction2">
    Install the toolbox and its dependencies using conda and pip:
    ```bash
    conda create --name openmmlab python=3.8 -y
    conda activate openmmlab
    pip install -U openmim
    mim install mmengine mmcv
    pip install mmaction2
    ```
  </Step>
  <Step title="Download a pre-trained model">
    Use the OpenMIM package manager to fetch a checkpoint and its config:
    ```bash
    mim download mmaction2 --config tsn_imagenet-pretrained-r50_8xb32-1x1x3-100e_kinetics400-rgb --dest .
    ```
  </Step>
  <Step title="Run inference on a video">
    Use the Python API to classify actions in any video file:
    ```python
    from mmaction.apis import init_recognizer, inference_recognizer

    config = 'tsn_imagenet-pretrained-r50_8xb32-1x1x3-100e_kinetics400-rgb.py'
    checkpoint = 'tsn_imagenet-pretrained-r50_8xb32-1x1x3-100e_kinetics400-rgb_20220906-cd10898e.pth'

    model = init_recognizer(config, checkpoint, device='cuda:0')
    result = inference_recognizer(model, 'demo/demo.mp4')
    print(result.pred_score.topk(5))
    ```
  </Step>
</Steps>
